3. EPS (Enterprise Project Structure)

The EPS is your project portfolio hierarchy. Think of it like folders.

Create EPS Node
  1. Go to Enterprise ‚Üí Enterprise Project Structure
  2. Click Add
  3. Name: 2025 Capital Program
  4. Responsible Manager: John Doe
Create New Project
Project ID: PRJ-1001
Project Name: New Office Building
Planned Start: 01/01/2025
Must Finish By: 12/31/2025
Rule: Always set Must Finish By date for deadline-driven projects.

5. Activities: The Heart of P6
Activity Types
Task Dependent
Default. Use for normal work.
Resource Dependent
Use when crew size matters.
Level of Effort (LOE)
For ongoing support (PM, QA).
Milestone
Start/Finish. Duration = 0.
Add Activity
  1. Go to Activities view
  2. Press Insert
  3. ID: A1000 (auto or manual)
  4. Name: Submit Design Package
  5. Duration: 5d
  6. Assign to WBS: 1.1
üí° Naming Convention: [Verb] + [Noun] ‚Üí "Install Rebar", "Pour Foundation"

Create Resource
Resource ID: LAB-001
Name: General Labor
Type: Labor
Default Units/Time: 8h/d
Cost/Unit: $45.00/h
Assign Resource
  1. Select activity
  2. Go to Resources tab (bottom)
  3. Click Add Resource
  4. Select LAB-001
  5. Budgeted Units: 40h
Golden Rule: Always assign Roles first, then replace with actual Resources later.
